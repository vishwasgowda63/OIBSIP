# EDA on Menu Dataset

## Objective
Perform Exploratory Data Analysis on menu nutrition dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Analysis Performed
- Data Cleaning
- Statistical Analysis
- Calories Analysis
- Sodium Analysis
- Correlation Heatmap
- Protein Distribution

## Insights
- Some food categories have high calories.
- Sodium levels are high in several menu items.
- High calorie foods contain more fat.
- Heatmap shows relationships among nutrition values.

## Conclusion
The project provides useful insights into nutritional patterns and food health analysis.

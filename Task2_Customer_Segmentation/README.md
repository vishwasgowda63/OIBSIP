# Customer Segmentation Analysis

## Objective
Analyze customer behavior and spending patterns using data analytics techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Analysis Performed
- Data Cleaning
- Income Distribution Analysis
- Age Distribution Analysis
- Spending Pattern Analysis
- Correlation Heatmap
- Customer Insights

## Insights
- Customers with higher income spend more.
- Wine and meat products are highly purchased.
- Most customers belong to middle-age group.
- Spending patterns help improve marketing strategies.

## Conclusion
Customer segmentation helps businesses understand customer behavior and improve targeted marketing decisions.
